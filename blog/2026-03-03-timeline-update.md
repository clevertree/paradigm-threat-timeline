## New Feature: Browser Mode

The Timeline now includes a **Browser** view mode that lets you navigate the entire repository as a file tree — content articles, blog posts, investigations, and documentation — all viewable directly on the website.

**Deep linking** is supported: any document can be linked directly using the format:
```
https://paradigm-threat.net/timeline?view=browser&path=<file-path>
```

This is especially useful for linking to **investigation files**, which are not part of the book but contain the primary research behind the timeline.

**13.06.00** — In 1865, James Clerk Maxwell published equations that power every electrical device on Earth. This article traces the suppression from Heaviside through the Michelson-Morley experiment to Einstein's relativity.
